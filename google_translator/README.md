# 🌐 Language Translator App

A simple and interactive desktop application built using Python that translates text between multiple languages in real time.

---

## 🚀 Features

* 🌍 Translate text between multiple languages
* 🖥️ User-friendly GUI built with Tkinter
* ⚡ Fast translation using Google Translate API
* 🔽 Dropdown selection for source and destination languages
* 🧹 Auto-clears and updates translated text

---

## 🛠️ Tech Stack

* **Language:** Python
* **GUI Library:** Tkinter
* **Translation API:** googletrans

## ▶️ How to Run

### 1. Clone the repository

```
git clone https://github.com/nandini2217/Python-Projects.git
cd Python-Projects/translator-app
```

### 2. Install dependencies

```
pip install googletrans==4.0.0-rc1
```

### 3. Run the application

```
python main.py
```

---

## 💡 How It Works

1. Enter text in the source box
2. Select source and destination languages
3. Click **Translate**
4. View translated output instantly
